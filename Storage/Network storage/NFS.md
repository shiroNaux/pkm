# Abstract

NFS (Network file system) là 1 giao thức (protocol) sử dụng trong các kết nối phục vụ lưu trữ, chia sẻ file, ... giữa các máy trong network.

Ngoài NFS, còn có 1 số giao thức khác cũng có mục đích tương tụ: SMB, AFP, ...
