# Introduction

Dynamic [[SQL]] là 1 tính năng của [[Oracle]], cho phép người dùng viết các câu [[sql|query]] dưới dạng generic, và rồi các caua query đó sẽ được render lúc runtime. Nó tương tụ như các phương pháp templating.
