# Abstract
- Để backup dữ liệu của 1 database trong [[PostgreSQL]] server, sử dụng `pg_dump`
- Để backup toàn bộ [[PostgreSQL]], sử dụng `pg_dumpall`

# Using

```
pg_dump database_name > file-name.sql
```
